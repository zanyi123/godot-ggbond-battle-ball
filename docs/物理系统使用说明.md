# 物理系统使用说明

---

## 系统概览

### 组成部分

| 组件 | 文件 | 职责 |
|------|------|------|
| **场地物理管理器** | `field_physics_manager.gd` | 管理全局 μ 和 e |
| **击退物理计算** | `knockback_physics.gd` | 计算击退距离和速度 |
| **球的物理量** | `ball.gd` | 存储球质量和弹性 |
| **球员击退** | `player.gd` | 读取摩擦，应用击退 |

---

## 调用链路

### 完整调用链

```
游戏启动
  ↓
BattleManager._ready()
  ↓
BattleManager._setup_field_physics_manager()
  ↓
创建 FieldPhysicsManager 节点
  ↓
挂载到 BattleManager
  ↓
对局开始
  ↓
Player.take_damage(球击中)
  ↓
Player._apply_knockback()
  ↓
Player._get_field_friction() ← 查询 FieldPhysicsManager
  ↓
返回 μ（默认1.0，或被技能修改）
  ↓
KnockbackPhysics.calculate_knockback(μ, 僵直时间)
  ↓
返回 distance 和 initial_velocity
  ↓
Player 匀减速动画
```

# 获取管理器

## 功能说明

### 1. 摩擦系数 μ

**默认值：** 1.0（标准地面）

**效果：** 影响球员击退距离

**范围建议：**
| 值 | 描述 | 击退距离变化 |
|---|------|-------------|
| 0.3~0.6 | 超滑/冰面 | 距离翻倍以上 |
| 0.7~0.9 | 湿滑路面 | 距离增加 |
| 1.0 | 标准地面 | 基准距离 |
| 1.1~1.4 | 粗糙地面 | 距离减少 |
| 1.5~1.7 | 泥潭 | 距离减半 |
| 1.8~2.0 | 粘胶 | 距离很短 |

**示例：**
```gdscript
# 冰面击退
field_physics.set_friction(0.5, "ice_skill", 10.0)
# 一段击退距离从 100px → 200px
```

---

### 2. 弹性系数 e

**默认值：** 0.0（无反弹）

**效果：** 影响球碰撞反弹（预留接口）

**范围建议：**
| 值 | 描述 |
|---|------|
| 0.0 | 无反弹（球撞墙停止或出界） |
| 0.3 | 弱反弹（仅30%速度反弹） |
| 0.5 | 中等反弹（一半速度反弹） |
| 0.8 | 强反弹（80%速度反弹） |
| 1.0 | 完全反弹（100%速度反弹） |

**注意：** 当前只存储值，碰撞系统未实现

---

### 3. 击退距离计算

**公式：**
```
d = (d_baseline / μ) × M_skill + offset
v = 2 × d / t_stagger
```

**参数：**
- `d_baseline` = 100px（一段）或 200px（二段）
- `μ` = 当前摩擦系数
- `M_skill` = 技能倍率（默认1.0）
- `offset` = 技能固定加成（默认0.0）
- `t_stagger` = 僵直时间（由韧性决定）

**示例：**
```gdscript
# 标准击退（μ=1.0, 僵直0.4s）
var result = KnockbackPhysics.calculate_knockback("knockback1", 1.0, 0.4)
# result.distance = 100px
# result.initial_velocity = 500px/s

# 冰面击退（μ=0.5, 僵直0.4s）
var result = KnockbackPhysics.calculate_knockback("knockback1", 0.5, 0.4)
# result.distance = 200px
# result.initial_velocity = 1000px/s
```

## 常见问题

### Q1: 击退距离没有变化？

**可能原因：**
1. FieldPhysicsManager 未创建（查看控制台是否有创建日志）
2. 摩擦系数仍然是 1.0
3. 查看路径是否正确

**解决方法：**
```gdscript
var battle = get_node("/root/BattleManager")
print("BattleManager 存在: ", battle != null)

if battle:
    var field_physics = battle.get_node("FieldPhysicsManager")
    print("FieldPhysicsManager 存在: ", field_physics != null)
    
    if field_physics:
        print("当前摩擦: ", field_physics.get_friction())
```
