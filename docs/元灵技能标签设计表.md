# 元灵技能类型标签设计表（完整版）

## 设计原则
1. **大标签**：作用类型分类（对球/对场地/对球员）
2. **小标签**：对称设计，进攻↔防守对等
3. **标签编码**：统一格式便于系统调用

## 六、标签扩展规则

1. **组合标签**：多个标签可组合使用，实现复杂效果
2. **条件触发**：标签可添加触发条件（如体力<30%时）
3. **持续效果**：标签可设置持续时间（瞬间/持续/永久）
4. **层级叠加**：同类标签可叠加或覆盖（根据优先级）
5. **目标指定**：标签可指定作用目标（自身/队友/敌人/全局）

---

## 七、系统实现要点

1. **标签注册表**：维护所有标签的代码映射表
2. **效果处理器**：每个标签对应一个处理函数
3. **触发检测器**：检测技能使用并调用对应标签
4. **效果堆栈**：管理场上所有活跃效果
5. **冲突解决**：处理标签冲突和优先级
6. **UI反馈**：向UI层发送效果生效消息
