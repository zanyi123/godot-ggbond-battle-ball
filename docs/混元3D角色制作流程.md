# 混元3D → Godot 动画角色流程

## 一句话总览

```
混元3D生成外观 → Blender清理 → Mixamo绑骨+动画 → 导入Godot
```

> 混元3D **只产出**：模型外形（Mesh）+ 贴图（Texture）
> 骨骼、权重、动画需要后续步骤补上

---

## 第1步：混元3D 生图（当前步骤）

- 网址：`https://3d.hunyuan.tencent.com/`
- 模式：**图生3D**（上传参考图 → 生成3D模型）
- 下载格式：**GLB**（Godot友好）

### 参考图要求

| 项目 | 建议 |
|------|------|
| 视角 | 纯正面 |
| 背景 | 白色/纯色 |
| 姿势 | T-Pose 或 A-Pose（双臂张开） |
| 风格 | 卡通/Q版生成效果最好 |
| 分辨率 | ≥ 512×512 |

---

## 第2步：Blender 清理模型

- 导入GLB → 检查面数（建议 < 1万面）→ 减面 → 导出FBX

---

## 第3步：Mixamo 绑骨 + 动画

- 网址：`https://www.mixamo.com`（Adobe免费）
- 上传FBX → 自动绑骨 → 选动画 → 下载带骨骼动画的FBX
- ⚠️ Mixamo 只适合**人形角色**，非人形需在Blender手动绑骨

### 决竞球需要的动画

| 动画 | 用途 |
|------|------|
| idle | 待机 |
| run | 跑动 |
| kick | 踢球 |
| tackle | 铲球 |
| catch | 扑救 |
| hurt | 受伤 |
| celebrate | 庆祝 |

---

## 第4步：导入 Godot 4

1. Blender 导出 GLB（勾选 Skins + Animations + Materials）
2. 放入项目 `assets/` 目录
3. 场景中添加 AnimationTree 控制动画切换
