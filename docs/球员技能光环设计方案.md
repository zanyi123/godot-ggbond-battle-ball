# 对球员元灵技能 - UI激活光环设计

## 设计目标
当玩家激活对球员的技能（如自身属性加成、负面效果等）时，需要在球员身上显示明显的视觉反馈，与球的光环系统形成统一的视觉语言。

---

## 方案对比

### 方案1：球员光环（推荐）⭐

**设计思路：**
- 在球员节点下方添加光环 Sprite2D（类似球光环）
- 持球时显示球光环 + 玩家光环（如果激活了球和球员两种技能）
- 光环位置：球员脚下（z_index = -2）
- 大小：比球光环稍大（scale = 2.5）
- 颜色：根据技能元素类型

**优点：**
- 视觉效果统一，玩家容易理解
- 与球光环形成呼应
- 可以同时显示球和球员技能状态

**缺点：**
- 可能与场地背景冲突

---

### 方案2：状态指示器增强

**设计思路：**
- 扩展现有 `state_indicator` (ColorRect)
- 技能激活时改变颜色 + 添加发光效果
- 在顶部/底部添加一个小光环圈

**优点：**
- 复用现有组件，实现简单
- 不会增加太多性能开销

**缺点：**
- 视觉效果不够明显
- 与球光环风格不统一

---

### 方案3：粒子特效流

**设计思路：**
- 使用 GPUParticles2D 在球员周围创建粒子流
- 根据元素类型使用不同粒子效果
  - 金刚：金色粒子向上飘浮
  - 雷火：红色粒子环绕
  - 冰雪：蓝色冰晶粒子

**优点：**
- 视觉效果炫酷
- 区分度高

**缺点：**
- 性能开销较大
- 实现复杂度高

---

### 方案4：外边框发光（组合方案）

**设计思路：**
- 玩家精灵周围添加发光边框
- 使用 BackBufferCopy + CanvasItemMaterial 实现发光效果
- 持球时：球光环 + 边框发光
- 不持球时：光环 + 边框发光

**优点：**
- 视觉效果明显
- 不会影响球员本身外观

**缺点：**
- 技术实现复杂

## 推荐实现方案：球员光环 + 状态指示器

### 整体设计

**状态优先级：**
```
持球 + 球技能激活 → 球光环（主）+ 玩家光环（次）
持球 + 球员技能激活 → 球无光环 + 玩家光环
不持球 + 球员技能激活 → 玩家光环
```

**光环视觉设计：**
```
       [球员]
    ↙         ↘
  ╔═════════════╗  ← 光环 (脉冲动画)
  ║             ║
   ║           ║
    ╚═══════════╝
```

---

### 实现架构

#### 1. Player 节点结构
```
Player (CharacterBody2D)
├── Sprite2D (球员外观)
├── CollisionShape2D
├── Area2D (检测球)
│   └── CollisionShape2D
├── ColorRect (state_indicator)
└── Sprite2D (skill_aura) ← 新增：技能光环
```

#### 2. 光环属性
| 属性 | 值 |
|------|-----|
| z_index | -2 (在球员下方，背景上方) |
| scale | (2.5, 2.5) |
| 位置 | (0, 20) - 脚下位置 |
| 脉冲速度 | 2.0 |
| 脉冲幅度 | 1.0 ~ 1.2 |

#### 3. Ball 节点结构
```
Ball (Area2D)
├── CollisionShape2D
├── ColorRect (ball_visual)
├── ColorRect (ball_shadow)
└── Sprite2D (skill_aura) ← 已有：球技能光环
```

### 代码实现

#### Player 新增变量
```gdscript
# 技能光环（显示对球员的技能）
var skill_aura: Sprite2D = null
var active_player_skill_data: Dictionary = {}
const PLAYER_AURA_PULSE_SPEED: float = 1.5
var player_aura_pulse_time: float = 0.0
```

#### Player 新增方法
```gdscript
# 创建球员光环
func _create_skill_aura() -> void:
    skill_aura = Sprite2D.new()
    skill_aura.name = "SkillAura"
    skill_aura.z_index = -2  # 在球员下方
    skill_aura.position = Vector2(0, 20)  # 脚下位置
    skill_aura.visible = false
    add_child(skill_aura)

# 显示玩家技能光环
func show_skill_aura(skill: Dictionary) -> void:
    if skill_aura == null:
        _create_skill_aura()

    active_player_skill_data = skill

    var element: String = skill.get("element", "")
    var aura_color: Color = _get_element_color(element)

    var texture := _create_aura_texture(aura_color)
    skill_aura.texture = texture
    skill_aura.modulate = aura_color
    skill_aura.scale = Vector2(2.5, 2.5)
    skill_aura.visible = true

# 清除玩家技能光环
func clear_skill_aura() -> void:
    if skill_aura:
        skill_aura.visible = false
    active_player_skill_data = {}

# 更新光环动画
func _process_player_aura(delta: float) -> void:
    if skill_aura and skill_aura.visible:
        player_aura_pulse_time += delta * PLAYER_AURA_PULSE_SPEED
        var pulse = 1.0 + 0.2 * sin(player_aura_pulse_time)
        skill_aura.scale = Vector2(2.5 * pulse, 2.5 * pulse)
```

#### Player 修改方法
```gdscript
func _process(delta: float) -> void:
    # ... 现有代码 ...

    # 更新技能光环动画
    _process_player_aura(delta)
```

### 技能激活逻辑

#### 判断技能类型
```gdscript
func _get_skill_target_type(skill_id: String) -> String:
    """返回 'ball' / 'player' / 'field'"""
    var skill_data = DataManager.get_skill_by_id(skill_id)
    if skill_data.is_empty():
        return "unknown"

    var tags = skill_data.get("tags", [])
    for tag_id in tags:
        if tag_id.begins_with("ball_"):
            return "ball"
        elif tag_id.begins_with("player_"):
            return "player"
        elif tag_id.begins_with("field_"):
            return "field"

    return "unknown"
```

#### 统一激活接口
```gdscript
func set_active_skill(skill_id: String) -> void:
    """统一激活技能接口"""
    var target_type = _get_skill_target_type(skill_id)
    var skill_data = DataManager.get_skill_by_id(skill_id)

    match target_type:
        "ball":
            # 球技能：如果持球，更新球光环
            _active_skill_id = skill_id
            if is_carrying_ball:
                _update_ball_skill_aura()
        "player":
            # 球员技能：显示玩家光环
            active_player_skill_data = skill_data
            show_skill_aura(skill_data)
        "field":
            # 场地技能：预留（可能需要标记区域）
            pass
        "unknown":
            print("[Player] 未知技能类型: %s" % skill_id)
```

```gdscript
var _aura_texture_cache: Dictionary = {}

func _get_cached_aura_texture(color: Color) -> Texture2D:
    var color_key = str(color.to_html())
    if _aura_texture_cache.has(color_key):
        return _aura_texture_cache[color_key]

    var texture = _create_aura_texture(color)
```

## 扩展功能

### 多技能光环叠加

```gdscript
# 使用多个光环节点显示不同元素技能
var skill_auras: Array[Sprite2D] = []
var active_skills: Array[Dictionary] = []

func add_skill_aura(skill: Dictionary) -> void:
    var aura = Sprite2D.new()
    var color = _get_element_color(skill.get("element", ""))
    aura.texture = _get_cached_aura_texture(color)
    aura.modulate = color
    aura.rotation = randf() * TAU  # 随机旋转角度
    add_child(aura)
    skill_auras.append(aura)
    active_skills.append(skill)
```
