# 决竞球 Battle Ball - 项目结构说明

## 目录结构

```
battle-ball/
├── project.godot                 # Godot 项目配置
├── docs/
│   └── 问题提问日志.md             # 设计问答记录
├── data/                         # 数据文件（JSON，方便开发者快速修改）
│   ├── characters/
│   │   └── characters.json       # 角色数据（6个球员）
│   ├── spirits/
│   │   ├── spirits.json          # 元灵数据（6个元灵）
│   │   └── elements.json         # 元素克制关系
│   └── skills/
│       └── skills.json           # 技能数据（所有技能）
├── scenes/                       # Godot 场景文件
│   ├── main/
│   │   └── main_menu.tscn        # 主菜单
│   ├── battle/
│   │   └── battle_arena.tscn     # 比赛场景
│   ├── prep/                     # 备战场景（待建）
│   └── results/                  # 结算场景（待建）
├── scripts/
│   ├── core/
│   │   ├── data_manager.gd       # 数据加载单例（Autoload）
│   │   └── game_manager.gd       # 比赛状态管理单例（Autoload）
│   ├── battle/
│   │   ├── player.gd             # 球员逻辑（移动、体力、技能、韧性）
│   │   ├── ball.gd               # 球逻辑（飞行、碰撞、技能注入）
│   │   ├── battle_manager.gd     # 比赛主控制器（场景管理）
│   │   ├── battle_hud.gd         # 比赛HUD（球员栏、比分、计时）
│   │   └── input_manager.gd      # 输入管理（发球预瞄、切换球员）
│   ├── ai/                       # AI系统（待建）
│   ├── data/                     # 数据处理脚本（待建）
│   └── ui/
│       └── main_menu.gd          # 主菜单逻辑
└── assets/                       # 美术资源（按分类存放）
    ├── characters/               # 球员美术
    │   ├── avatars/              # 头像
    │   ├── sprites/              # 精灵图
    │   └── animations/           # 动画
    ├── spirits/                  # 元灵美术
    │   ├── avatars/
    │   ├── sprites/
    │   └── effects/
    ├── icons/                    # 图标
    │   ├── skills/               # 技能图标
    │   ├── items/                # 道具图标
    │   ├── buffs/                # Buff图标
    │   └── ui/                   # UI图标
    ├── field/                    # 场地资源
    ├── ui/                       # UI美术
    │   ├── backgrounds/
    │   ├── frames/
    │   ├── buttons/
    │   └── popup/
    ├── audio/                    # 音频
    │   ├── bgm/
    │   ├── sfx/
    │   └── voice/
    ├── effects/                  # 特效
    │   ├── particles/
    │   ├── projectiles/
    │   └── aura/
    └── misc/                     # 其他
```
