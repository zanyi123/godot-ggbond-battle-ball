## 一、元灵技能球类标签 ID 对齐

### 问题
`tags_registry.json` 注册表中的球类标签 ID 与 `spirit_tag_effect_handler.gd` 处理器的 match 分支 ID 不一致：
- 注册表拆分为 `_pct` / `_flat` 后缀（如 `ball_dmg_up_pct`、`ball_dmg_up_flat`）
- 处理器仍用旧版合并名（如 `ball_dmg_up`，靠参数 `value_type` 区分）
- 导致前8个数值标签运行时全部走 `_` 默认分支，静默失败

### 修复
**文件**：`scripts/systems/spirit_system/spirit_tag_effect_handler.gd`

- match 分支从旧6个 → 新17个，与注册表 BALL 标签一一对应
- 4个合并函数拆分为8个独立函数：
  - `_apply_ball_dmg_up` → `_apply_ball_dmg_up_pct` + `_apply_ball_dmg_up_flat`
  - `_apply_ball_dmg_down` → `_apply_ball_dmg_down_pct` + `_apply_ball_dmg_down_flat`
  - `_apply_ball_speed_up` → `_apply_ball_speed_up_pct` + `_apply_ball_speed_up_flat`
  - `_apply_ball_speed_down` → `_apply_ball_speed_down_pct` + `_apply_ball_speed_down_flat`
- 移除注册表中不存在的 `ball_armor` match 分支（函数保留备用）
- 编号对齐注册表 code 字段（01-08 数值类，09-15 飞行行为类，16-17 范围类）

### 四链路一致性
```
skills.json tags[] → tags_registry.json id → handler match → handler 函数名
全部一致 ✅
```

## 二、追踪球（ball_tracking）链路修复

### 问题
1. 追踪球飞到 `max_flight_distance` 就落地消失，无法持续追踪
2. 击中球员后球总是回到攻击者手上，不是被击中者

### 修复
**文件**：`scripts/battle/ball.gd`

- `_physics_process`：追踪球跳过 `max_flight_distance` 距离限制，只在命中球员或出界时停止
- `_on_body_entered` 球权判断逻辑重写：

| 命中情况 | 追踪球球权 | 普通球球权 |
|---|---|---|
| 击败敌方球员 | 受击者同队最近存活球员 | 攻击者手上 |
| 韧性弹飞（ball_fly） | 球偏转继续追踪，不停 | 球偏转后继续飞（有距离上限） |
| 击退不弹飞 | **受击者接球** | 攻击者手上 |
| 普通命中 | **受击者接球** | 攻击者手上 |
| 同队队友 | 队友接球 | 队友接球 |
| 穿透+追踪 | 穿透优先继续飞 | 穿透优先继续飞 |

## 三、球碰撞检测修复（核心 Bug）

### 问题
1. 球的碰撞依赖 `body_entered` 信号（Area2D），不可靠，球可能穿过球员不触发
2. 球飞到距离上限落地走 `_on_ball_stopped()` → 直接给最近球员，**完全没有伤害判定**
3. 表现：对手投球→球停在玩家附近→玩家直接"捡球"→不掉血→韧性系统不触发

### 修复
**文件**：`scripts/battle/ball.gd`

| 改动 | 说明 |
|---|---|
| `var _hit_player_ids` | 命中追踪字典，防止双重命中（body_entered + 距离检测同时触发） |
| `_check_player_collision_distance()` | 每帧距离检测，范围 = 球半径(14) + 球员半径(28) + 球速×帧时间，防止高速穿透 |
| `_on_ball_stopped()` | 落地前检查60px内有没有球员，有则视为命中走 `_on_body_entered`（含伤害+韧性），没有才按半场分配球权 |
| `_get_all_players_array()` | 提取公共函数，统一获取所有球员数组 |
| `launch()` | 每次发球重置 `_hit_player_ids` |

### 修复后韧性系统生效链路
```
球飞行中(距离检测) / 球落地(停球检测)
  → _on_body_entered(player)
    → 同队队友 → 接球，不扣血
    → 对方球员 → take_damage(ball_damage)
      → is_ready_to_catch?
        → 是：韧性伤害衰减 + 效果判定（击退/弹飞/球弹飞）
        → 否：全额伤害 - 防御抗力 + 僵直
```
