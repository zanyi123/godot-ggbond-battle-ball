## 🔴 核心玩法机制

### Q1. 球的物理与飞行
- **直线还是弧线？** 都有，备战时可设置
- **非目标球员能否拦截？** 可以，只要处于待接球状态。未处于待接球状态的球员被球击中则球反弹回攻击者手中
- **几个球？** 全局1个实球

### Q2. AI 队友
- **操控方式？** 玩家操控1人，另2人AI控制；对方全AI
- **AI 行为逻辑？** 备战时设置（跟防/跟球/自保/团队策略）
- **多人模式？** 先做AI模式，双人往后放

### Q3. 防御接球机制
- **判定方式？** 需要球员主动进入待接球状态（右键），涉及时机
- **体力消耗？** 接球时扣体力 = 对方球的攻击值（不使用元灵技能、不触发韧性时）
- **成功接球后？** 球归该球员
- **韧性触发条件？** 韧性只在待接球状态才发挥作用（R1修正）
- **韧性效果？** 高韧性→高概率减伤+球弹飞（随机方向）；低韧性→低概率减伤+高概率被击退/击飞出界

### Q4. 体力值系统
- **范围？** 0-100（最高上限），每个球员原始数据不同
- **伤害计算？** 基础攻击值（黑属性）+ 元灵克制
- **自动回复？** 不会，类似血条

### Q5. 球员被击败
- **淘汰？** 不淘汰。对方得分，但球员继续留在场上
- **击败后效果？** 元灵能量上限减半、恢复减半、攻击力减半
- **重复得分？** 被击败后不能再次从该球员身上得分
- **受伤下场？** 特殊情况需下场（待细化）

### Q6. 得分后球权
- **溢出 >= 5？** 球回到得分方淘汰发球手中
- **溢出 < 5？** 球归被得分方
- **发球位置？** 球员手上

## 🔴 补充澄清（追问轮）

### R1. 韧性完整流程
```
球飞向目标球员：
├─ 目标处于「待接球」状态 → 扣体力 + 韧性判定
│    ├─ 高韧性 → 高概率减伤 + 球弹飞（随机方向）
│    └─ 低韧性 → 低概率减伤 + 高概率被击退/击飞出界
│    接住球 → 球归该球员
└─ 目标不在「待接球」状态 → 被击中，球反弹回攻击者手中
```
注：韧性效果**只在待接球状态**才发挥作用。球弹飞方向为随机。

### R2. 非目标球员拦截
- 处于待接球状态的球员在球的飞行路径上均可拦截
- 拦截范围待定（需参考场地尺度，待场地图片确定）

### R3. 溢出 < 5 球权
- 球归被得分方

### R4. 球的攻击值计算
- **基础伤害** = 黑属性（基础攻击值）
- **发球力度** = 只决定飞行距离，不计算伤害
- **发球时** 生成一个预落点显示
- **元灵克制** = 双方交互时使用元灵技能（技能注入发球，技能接球）

### R5. 出界得分
- 被击退导致出界 → 对手得分
- 主动跑出界 → 也是对手得分（玩家自己的问题）
- 出界后处理 → 回到场内（位置重置）

## 🟡 元灵与技能

### Q7. 元灵携带
- **数量？** 每个球员带1个元灵
- **技能？** 3个主动技能 + 可选1个被动技能（最多4个）
- **技能选择？** 备战时选定，中场备战无法更改

### Q8. 元灵能量
- **范围？** 0-100，上限不超过100
- **回复方式？** 随时间自动回复 + 击中对手回复 + 被击中回复（具体数值待设定）

### Q9. 技能类别与示例
- **"对球"类** — 增伤、改轨迹、分裂、球速、减速、减伤等
- **"对球员"类** — 减速对手、减防、减伤、护盾等
- **"对场地"类** — 放置障碍、改变地形、冰面减速区域等

技能标签系统：技能按标签分类，方便调用计算

技能使用流程：使用技能 → 读取技能标签 → 该标签下所有元灵技能收录 → 读取技能数值 → 返回参与计算

### Q10. 克制关系效果
- 具体数值待定（先打基础框架）
- 金刚与梦幻互克。草木克大地，大地克冰雪，冰雪克雷火，雷火克草木

### R6. 技能解锁
- 消耗矿石解锁（与童话币机制相同的道具）
- 球员等级（训练）提升
- 元灵升级消耗资源（矿石）

### R7. 技能冷却与施法
- 有 CD
- 有前摇/施法时间

### R8. 初始6个元灵
- 6大元素各1个，名字参照原动画主角团名字

### R9. 天赋示例
- 超人强球员天赋：防御+5%（在场上时，基于基础血量的5%增加永久护盾，不再生）

## 🔵 球队角色分工

三个位置（备战时划分）：
- **主攻手**：利用技能与球技对对手进攻
- **防御手**：保护队友，防下对手攻击
- **辅助手**：帮助队友进攻/防御

球员面板属性：基础体力值（红）、防御力值（黄）、移速（蓝）、基础攻击值（黑）、韧性度（0-80）、描述、元灵偏好
