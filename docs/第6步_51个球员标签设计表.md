# 51个球员标签设计表

## 所有可选选项

### ①属性 — 改哪个属性？

| 属性 | 作用 |
|------|------|
| 攻击力 | 球的伤害基数 |
| 防御力 | 伤害减免（防御力×防御因子=抗伤） |
| 移动速度 | 每帧位移距离 |
| 韧性 | 查表决定伤害衰减/僵直时间/击退概率 |
| 最大能量 | 能量上限 |

改法：提升(%) / 降低(%) / 提升(固定) / 降低(固定)

---

### ②状态 — 点哪盏灯？

| 灯 | 亮了会怎样 | 对应标签 |
|----|-----------|---------|
| 无敌灯 | 受到伤害变0 | player_invincible |
| 易伤灯 | 受到伤害×倍率（你要填倍率） | player_vulnerable |
| 隐身灯 | 敌方索敌/追踪/AOE跳过你，但球照样能撞到你 | player_stealth |
| 定身灯 | 不能移动，可以投球和技能 | player_root |
| 眩晕灯 | 不能移动、不能投球、不能技能 | player_stun |
| 免控灯 | 眩晕/沉默/缴械/定身这4盏灯点不上去 | player_cc_immune |
| 沉默灯 | 不能使用技能 | player_silence |
| 缴械灯 | 不能投球 | player_disarm |

---

### ③持续 — 每秒干什么？

| 类型 | 效果 | 对应标签 |
|------|------|---------|
| 每秒回复 | 每秒恢复X体力，不超过上限 | player_hp_regen |
| 每秒掉血 | 每秒扣X体力，到0击败（无敌可挡） | player_hp_dot |

---

### ④折扣 — 打什么折？

| 折扣类型 | 效果 | 对应标签 |
|---------|------|---------|
| 消耗折扣 | 技能能量消耗×倍率 | player_spirit_cost_down |
| 消耗涨价 | 技能能量消耗×倍率 | player_spirit_cost_up |
| CD缩短 | 技能冷却×倍率 | player_spirit_cd_down |
| CD延长 | 技能冷却×倍率 | player_spirit_cd_up |
| 效果翻倍 | 下次技能数值标签的value×2（一次性） | player_spirit_double |
| 效果减半 | 下次技能数值标签的value×0.5（一次性） | player_spirit_half |

---

### ⑤动作 — 干什么？

| 动作 | 对应标签 | 对谁 |
|------|---------|------|
| 回复最大体力% | player_hp_heal_pct | 自己/敌方/队友 |
| 扣最大体力% | player_hp_damage_pct | 自己/敌方/队友 |
| 回复固定体力 | player_hp_heal_flat | 自己/敌方/队友 |
| 扣固定体力 | player_hp_damage_flat | 自己/敌方/队友 |
| 回复最大能量% | player_energy_gain_pct | 自己/敌方/队友 |
| 扣最大能量% | player_energy_cost_pct | 自己/敌方/队友 |
| 回复固定能量 | player_energy_gain_flat | 自己/敌方/队友 |
| 扣固定能量 | player_energy_cost_flat | 自己/敌方/队友 |
| 技能使用次数+N | player_spirit_uses_up | 自己 |
| 关掉定身灯（解控） | player_unroot | 目标 |
| 拍灭敌方所有隐身灯 | player_reveal | 敌方全体 |
| 传送到目标位置 | player_teleport | 自己 |
| 返回上一位置 | player_return | 自己 |
