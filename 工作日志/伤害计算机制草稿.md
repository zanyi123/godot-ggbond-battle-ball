# 伤害计算机制

## 基础伤害公式

```
被攻击后新血量 = 体力值 + 防御抗力 - 对方球攻击 × (1 - 韧性伤害衰减百分比)
```

### 防御抗力

```
防御抗力 = 防御值 × 防御因子（0.1~0.2，因球员而异）
```

| 球员 | 防御值 | 防御因子 | 防御抗力 |
|------|--------|----------|----------|
| 猪猪侠 | 60 | 0.15 | 9.0 |
| 超人强 | 80 | 0.20 | 16.0 |
| 菲菲 | 50 | 0.12 | 6.0 |
| 小呆呆 | 65 | 0.15 | 9.75 |
| 波比 | 55 | 0.10 | 5.5 |
| 迷糊老师 | 70 | 0.18 | 12.6 |

## 韧性计算表（rd = 韧性值）

### 一、韧性伤害衰减百分比（与效果同时发生）

| rd 区间 | 衰减公式 | 衰减百分比 |
|---------|----------|-----------|
| [90, 100] | 0.5 × rd | 45%~50% |
| [80, 90) | 0.4 × rd | 32%~36% |
| [60, 80) | 0.3 × rd | 18%~24% |
| [40, 60) | 0.2 × rd | 8%~12% |
| [30, 40) | 0.1 × rd | 3%~4% |
| [0, 30) | 0 | 0% |

### 二、效果发生概率（与衰减同时发生，三选一）

| rd 区间 | 击退+弹飞 | 球弹飞 | 击退 |
|---------|-----------|--------|------|
| [0, 30) | 0.3 | 0.45 | 0.25 |
| [30, 70) | 0.2 | 0.4 | 0.4 |
| [70, 100] | 0.1 | 0.4 | 0.5 |

**球弹飞**：保持攻击手原球速，方向随机偏转±90度，落地后离哪队近给哪队最近球员

**击退分段**：
- 一段击退：100px
- 二段击退：200px（100+100）

### 三、二段击退概率

```
二段击退概率 = 体力因子 × 元灵能量因子
一段击退概率 = 1 - 二段击退概率
```

| 体力比例 | 体力因子 |
|----------|----------|
| 0-30% | 0.6 |
| 30-60% | 0.3 |
| 60-100% | 0.1 |

| 剩余元灵能量 | 元灵能量因子 |
|-------------|-------------|
| 0-30% | 0.5 |
| 30-60% | 0.3 |
| 60-100% | 0.2 |

---

## 传球

传球力度只影响球的射程，不参与伤害公式。

---

## 注

- 元灵技能可以影响公式里的任何变量，以上为基本公式
- 韧度 = rd = 韧性值
- 待接球状态不触发韧性效果（正常接球）
