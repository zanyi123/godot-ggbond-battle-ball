# 2026-06-10b 工作日志

## 今日工作

### 1. BuffManager 接入 Player.gd ✅

之前 BuffManager 是独立模块，没接入 player.gd。现在完成了：

- player.gd 加 `_buffs` 字典 + `add_buff` / `remove_buff` / `_get_effective_value` / `_tick_buffs`
- `_physics_process` 三分支（击退/僵直/正常）都加 buff tick
- 属性读取全部走堆栈：
  - 速度：`_get_effective_value("speed", speed)`
  - 防御：`_get_effective_value("defense", defense) × defense_factor`
  - 韧性：`_get_effective_value("resilience", resilience)`（衰减率/僵直/击退概率）
- `take_damage` 中 `effective_amount`（无敌+易伤）已在第2步接入，防御/韧性 buff 在本步接入

### 2. Handler 51个标签函数全部填写 ✅

- 删除旧版80行空壳函数（全是 `pass`）
- 新增51个 match 分支路由
- 新增通用/专属函数：
  - `_apply_player_stat_buff` — ①属性类（20个标签共用）
  - `_apply_player_status` — ②状态类（7个标签共用，免控自动拦截）
  - `_apply_player_vulnerable` / `_apply_player_reveal` — 易伤/显形专属
  - `_apply_player_hp_*` — ⑤体力动作（4个即时 + 2个持续）
  - `_apply_player_energy_*` — ⑤能量动作（4个即时）+ ①能量属性（4个走buff）
  - `_apply_player_spirit_*` — ④折扣类（消耗/CD/倍率）
  - `_apply_player_teleport` / `_apply_player_return` — ⑤交互
  - `_get_player_targets` — 目标选择（self/enemies/allies/nearest_enemy）
  - `_get_all_enemies` — 含隐身者的敌方列表（用于显形）
- 全部标签支持 `_skill_mult` 效果倍率卡
- 全部标签支持 `target` 目标选择
