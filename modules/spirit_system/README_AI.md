# 元灵技能系统 - 模块上下文

## 模块名称
元灵技能系统 (Spirit Skill System)

## 模块目标
独立 UI 面板，管理元灵的选择、技能装备、技能解锁和元灵升级。

## 关键约定
- 主动技能最多3个，被动技能最多1个，固定不变
- 货币：元灵矿石（升级）、元灵水晶（解锁技能）
- 6种元素：金刚、大地、雷火、冰雪、草木、梦幻
- 每个元灵2-7个技能
- 升级=技能效果增强，不解锁新技能
- 技能解锁条件：元灵达到指定等级 + 消耗元灵水晶
